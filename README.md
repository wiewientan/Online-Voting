# Online Voting Web Application

This project is an online voting web application developed using JSP (Java Server Pages).

## Project Type
Java Web Application (JSP & Servlet)

## Development Tools
- NetBeans IDE
- Apache Tomcat
- MySQL Database

## How to Run
1. Extract the ZIP file
2. Open the project using NetBeans
3. Configure Apache Tomcat server
4. Import the database into MySQL
5. Run the project from NetBeans

> Note: This repository contains the source code in ZIP format.
> GitHub cannot automatically detect Java/JSP projects from ZIP files.
